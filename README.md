# 🎵 Mix Festival

O Mix Festival nasceu da ideia de unir diferentes estilos musicais em um único lugar, criando uma experiência cheia de música, cultura e diversão para todos os 
públicos.

Este projeto foi desenvolvido durante as aulas de desenvolvimento web com o objetivo de praticar HTML de forma criativa e dinâmica. O site apresenta a história do 
festival, programação dos shows, links para os artistas no Spotify, informações sobre ingressos, contato e localização do evento.

Ao longo do desenvolvimento, foram utilizados conceitos importantes como:
- Links internos e externos
- Âncoras de navegação
- Tabelas
- Organização de páginas
- Estrutura HTML5

O festival conta com atrações de diversos gêneros, como sertanejo, funk, pagode, forró, reggae, rock e MPB, tornando o projeto mais interativo e próximo de um evento
real.
